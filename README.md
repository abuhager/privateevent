طلاق نظام UniEvents: منصة ذكية لإدارة الفعاليات الجامعية 🚀

يسعدني أن أشارككم اليوم مشروعي لتخرج/للمادة (UniEvents)، وهو نظام ويب متكامل يهدف إلى تسهيل عملية تنظيم وحضور الفعاليات داخل الحرم الجامعي، وحل مشكلة الاكتظاظ والحجوزات اليدوية.

💡 فكرة المشروع: النظام ليس مجرد موقع لعرض الفعاليات، بل يحتوي على خوارزمية ذكية (Smart Booking Logic) تدير السعة المقعدية تلقائياً. إذا اكتمل العدد، يتم تحويل الطالب لقائمة الانتظار (Waitlist)، وبمجرد انسحاب أي مشارك، يقوم النظام آلياً بترقية أول شخص في الانتظار وتثبيت حجزه فوراً دون تدخل بشري.

🛠️ التقنيات المستخدمة:

Back-end: ASP.NET Core MVC (C#)

Database: SQL Server & Entity Framework Core

Front-end: HTML5, CSS3, Bootstrap

Security: BCrypt Hashing & Role-Based Authorization
